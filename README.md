# Quantum-Logic-Circiuit-Generator
This is a project for 'jokes' that i started when I got bored in class, and is not serious.

This is a project to draw logic diagrams for quantum computing via python turtle (only thing school computers can run) with the ability to save it in .eps format.

Using combinations or CNOT and CCNOT gates, 'standard' logic gates were able to be created, being:
  AND and NAND,
  XOR and XNOR,
  NOR AND OR,
  Buffer and NOT.

With these combinations, full adders were able to be rendered, and going one by one, the circuit was correct ( operation order being time, hence the diagram has logic flow from left to right ).
